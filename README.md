# Claude Code Configuration Management System

A comprehensive system to manage, share, and synchronize Claude Code configurations across multiple devices.

## Goal

Configure Claude Code once, deploy everywhere. This repository provides tools to version control and distribute:
- Global CLAUDE.md files
- Slash commands
- Claude personas
- Claude Skills
- Development environment configurations

## Target Use Case

Manage configurations across:
- Multiple Linux machines (Ubuntu, Fedora, Arch)
- iPhone (Claude Code web via Claude iOS app)
- Podman dev containers

## Approach

Hybrid configuration management combining:
- **Git + GitHub** for version control
- **Installation scripts** for deployment
- **chezmoi** for advanced templating (planned)
- **DevContainer Features** for container automation (planned)
